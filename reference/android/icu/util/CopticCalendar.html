CopticCalendar

public final class CopticCalendar
extends Calendar

java.lang.Object
   ↳ android.icu.util.Calendar
     ↳ android.icu.util.CopticCalendar

Implement the Coptic calendar system.

CopticCalendar usually should be instantiated using getInstance(ULocale) passing in a ULocale with the tag "@calendar=coptic".

See also:

  • Calendar
